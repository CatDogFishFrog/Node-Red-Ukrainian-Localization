# Node-Red-Ukrainian-Localization
Українська локалізація для інтерфейсу Node-Red. Зроблено для власного використання.

Просто скопіювати вміст теки `@node-red` до `\npm\node_modules\node-red\node_modules\@node-red`

Локалізація буде додана як нова і не замінить нічого. Проте, в інших мовах буде відображатися `languages uk-UA`, бо для цього вже треба змінювати їх власну локалізацію, але це не критично.

Особливості локалізацій:
- Далеко не все що зустрінеться в інтерфейсі перекладено! Однак, так і треба, бо не все повино бути перекладено, тож не страшно, та й це весь текст який був у модулі `@node-red`.
- До документації також додано невеличкі додаткові описи, не порушуючи основний текст.
- Текст для терміналу не перекладено (бо я так захотів, бо в терміналі текст повинен бути англійською...)
